## 标注规则

| 标记 | 含义 | 修改原则 |
| --- | --- | --- |
| `O` | 越南语翻译错误 | 修改 `vi` / `vietnamese` / 越南语提示字段，不改中文和拼音 |
| `X` | 词性错误 | 修改 `pos` / 词性显示字段，不改词条文本 |
| `✓` | 音频文件错误 | 核对并修正 normal/slow audio 路径或绑定 |
| `=` | 课名前添加级别和课文编号 | 修改课名显示/lesson title，不改词条内容 |
| `△` | 位置放错 | 从“词汇”移到“句子”或对应句子模块 |
| `><` | 词语重复 | 删除重复词条或合并重复显示 |
| `@` | 音频或答案绑定异常 | 核对音频、答案、句子绑定关系 |
