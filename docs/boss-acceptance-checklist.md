# 老板饭后验收清单

## 打开方式

本地地址：

```text
http://localhost:4173
```

## 1. 首页验收

- [ ] 页面能正常打开。
- [ ] 右上角有语言切换。
- [ ] 能看到 HSK 课程入口。
- [ ] 能看到 高频汉语 入口。
- [ ] 错题复习入口存在。
- [ ] 首页整体不遮挡、不溢出。

## 2. HSK 课程验收

- [ ] 进入 HSK 课程。
- [ ] 能看到 HSK2 / HSK3 / HSK4 / HSK5。
- [ ] 进入任意等级后，课程列表能显示对应 lesson。
- [ ] 点击 lesson 后能进入练习页。
- [ ] 训练顺序是从生词、短语到句子。

建议抽查：

- [ ] HSK2 第1课
- [ ] HSK3 任意 1 课
- [ ] HSK4 任意 1 课
- [ ] HSK5 任意 1 课

## 3. 高频汉语验收

- [ ] 进入 高频汉语。
- [ ] 能看到 12 个主题。
- [ ] 每个主题都能进入练习。
- [ ] 主题内容不是旧样例内容。

建议抽查主题：

- [ ] 问候与介绍
- [ ] 工作面试
- [ ] 课堂学习
- [ ] 点餐购物
- [ ] 电商职场

## 4. 越译中模式验收

- [ ] 页面显示越南语题干。
- [ ] 学生输入拼音。
- [ ] 拼音正确后显示对应汉字。
- [ ] `nihao / ni hao / 你好 / 你 好` 这类输入都能按规则判断。
- [ ] 未答对之前，下划线底部不显示答案提示。
- [ ] 答对当前格后，只显示当前格解释。

重点检查历史 Bug：

- [ ] `申请岗位` 显示为 `申请 + 岗位`，不是 `申 + 请 + 岗位`。
- [ ] `岗位要求` 显示为 `岗位 + 要求`，不是 `岗位 + 要 + 求`。
- [ ] 长句不会只有一条大下划线。

## 5. 听写模式验收

- [ ] 切换到听写模式后，页面不提前显示中文答案。
- [ ] 播放中文语音后，学生输入拼音。
- [ ] 未答前右侧记忆卡不泄题。
- [ ] 答错后不能跳过。
- [ ] 答对后可以进入下一题。

## 6. 音频验收

- [ ] 点击 播放，能听到中文语音。
- [ ] 点击 慢速，速度明显比普通播放慢。
- [ ] 生词、短语、句子朗读速度有层次。
- [ ] 输入拼音时有短促键盘音效。
- [ ] 答对音效和答错音效明显不同。
- [ ] 完成练习时有奖励反馈音效。

说明：浏览器普通话音色取决于电脑系统安装的中文语音包。当前程序已优先选择 `zh-CN` / `zh-Hans` 普通话声音。

## 7. UI 适配验收

电脑端：

- [ ] 顶部导航不显得空。
- [ ] 主训练区域不被底部按钮遮挡。
- [ ] 长越南语题干不会挡住输入格。
- [ ] 右侧记忆卡不挤压主训练区。

手机端：

- [ ] 首页能完整滚动。
- [ ] 课程卡片不溢出屏幕。
- [ ] 练习页输入格可点击。
- [ ] 底部按钮不遮挡输入内容。

## 8. 完成页与进度

- [ ] 完成一个 lesson / 主题后出现完成页。
- [ ] 今日学习目标能体现完成状态。
- [ ] 错题数量能记录。
- [ ] 收藏功能可用。
