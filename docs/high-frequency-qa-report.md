# 高频汉语模块 QA 报告

## 当前范围

- 主题数量：12
- 每个主题：10 个生词、30 个短语、4 个句子
- 总训练项：528
- 训练方式：听中文音频或看越南语提示，输入拼音或汉字，逐格判断

## 已完成检查

1. 结构完整性
   - 每个主题保持固定结构：10 word / 30 phrase / 4 sentence。
   - `index.html` 已加载 `lesson-high-frequency-topics.js`。

2. 分词与输入格
   - 短语和句子必须拆成多个 answer slot。
   - 高频模块非生词项不允许出现超过 5 个汉字的长输入格。
   - 全部 lesson 包不允许出现超过 6 个汉字的长输入格。

3. 词性检查
   - `申请岗位` 固定为：申请/verb + 岗位/noun。
   - `岗位要求` 固定为：岗位/noun + 要求/noun。
   - 高频动词已补充覆盖：出发、到达、打车、租房、搬家、维修、发烧、咳嗽、头疼、请假、休息、恢复、上班、下班、加班、汇报、负责、管理、处理、直播。

4. 防泄题规则
   - 未答对当前输入格之前，不显示词性、中文答案、越南语解释。
   - 当前格答对后，只显示当前格解释。
   - 整题完成后，显示完整记忆卡。

## 已修复问题

1. `申请` 被错误拆成 `申/请`。
2. `要求` 被错误拆成 `要/求`。

4. 未答前下划线底部提前显示提示。
5. 长句输入格可能退化为整句输入。
