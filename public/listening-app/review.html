验收检查项

□ 播放页：播放/暂停、上一句、下一句、重播当前句、demo mode 提示

□ 字幕：中文和拼音同步显示，当前句高亮明显

□ 重点词：点击后显示中文、拼音、越南语、收藏、打字练习入口

□ 跟读：任意句可进入模拟评分，很好/一般/需要练习

□ 收藏/打卡：收藏本期、收藏词汇、完成打卡后到我的页查看

□ Typing：由 typing_words 自动生成输入练习，显示正确率和完成状态

□ 会员：播放结束后看到自然的会员引导卡片

本地运行

运行：npm.cmd run dev -- --hostname 127.0.0.1 --port 3000。真实音频缺失时会自动进入 demo mode，不阻塞验收。

已知风险
