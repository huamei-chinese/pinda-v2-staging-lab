# 主播明天测试使用说明

## 测试版本

- 版本名称：Monday Test Version
- 稳定分支：`v1-test-monday`
- 项目类型：纯前端静态网页
- 登录状态：当前版本没有登录系统，不需要测试账号
- 敏感信息：当前前端未配置 API Key、密码、token

## 访问方式

### 方式 A：公网测试链接（推荐）

主播直接打开下面链接即可测试，不需要安装代码：

```text
https://huamei-chinese-practice-v1-monday.netlify.app/index.html
```

测试账号：

```text
当前 V1 没有登录系统，不需要账号和密码。
```

### 方式 B：本机浏览器访问

在项目目录运行：

```powershell
node server.js
```

本机打开：

```text
http://127.0.0.1:4173/index.html
```

### 方式 C：局域网访问（公网链接不可用时备用）

适合主播用手机或另一台电脑测试。要求测试设备和本机连接同一个 Wi-Fi。

1. 在项目目录运行：

```powershell
node server.js
```

2. 查看本机局域网 IP：

```powershell
ipconfig
```

找到 `WLAN` 或当前联网网卡下面的 `IPv4 Address`。

当前检测到的本机 IPv4：

```text
192.168.1.121
```

3. 主播访问地址：

```text
http://192.168.1.121:4173/index.html
```

如果 IP 改变，以当天 `ipconfig` 显示的 IPv4 为准。

## 测试范围

请重点测试：

- 首页是否正常显示
- HSK 课程入口是否正常
- HSK2/3/4/5 lesson 是否能进入
- 高频汉语 12 个主题是否能进入
- 越译中模式是否能输入拼音或汉字
- 听写模式是否能播放中文语音并输入拼音
- 正确、错误、完成音效是否正常
- 完成一题后是否显示“按 Enter / 空格继续”
- 手机端是否能正常显示和操作
- 电脑端是否能正常显示和操作

## 不需要测试的内容

本次不用测试：

- 登录注册
- 后台管理
- 用户账号系统
- 付费功能
- 数据云同步
- 老师端管理
- 新课程上传
- AI 自动生成题目

这些功能当前 V1 测试版没有包含。

## 发现问题如何记录

发现问题时，请记录以下信息：

1. 使用设备：手机 / 电脑
2. 浏览器：Chrome / Edge / Safari / 其他
3. 页面位置：首页 / HSK课程 / 高频汉语 / 训练页 / 听写页
4. 操作步骤：先点了什么、再输入了什么
5. 问题截图或录屏
6. 期望结果
7. 实际结果

示例：

```text
设备：手机
浏览器：Chrome
页面：HSK2 第1课 训练页
操作：输入 luyou 后按 Enter
问题：没有进入下一题
期望：完成后显示继续提示，再按 Enter 进入下一题
实际：停留在当前题
截图：已附
```

## 注意事项

- 测试时不要关闭运行 `node server.js` 的终端窗口。
- 手机访问失败时，先确认手机和电脑在同一个 Wi-Fi。
- 如果 Windows 防火墙弹窗，请允许 Node.js 在专用网络中通信。
- 如果 `4173` 端口被占用，可以临时换端口：

```powershell
$env:PORT=4174; node server.js
```

对应访问地址改为：

```text
http://本机IPv4:4174/index.html
```
